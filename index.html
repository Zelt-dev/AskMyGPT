О проекте

AskMyGPT – это современное кроссплатформенное чат-приложение, разработанное на Flutter, которое позволяет пользователям вести диалоги с различными AI-моделями. Оно сочетает в себе интуитивно понятный дизайн и широкие возможности управления чатами и шаблонами.

Проект был создан в рамках курса «Технологии программирования» в Международном университете МИТСО кафедры информационных технологий под руководством ИП Озерова Виктора Андреевича. Учебный проект выполнен для EPAM Systems в 2025 году.

Основные возможности

Кроссплатформенность

Работает на Android, iOS, Web и Desktop, обеспечивая единый пользовательский опыт на всех устройствах.

Мульти-чатовость

Создавайте, сохраняйте и группируйте чаты по датам для удобного управления диалогами.

Поддержка AI моделей

Выбирайте между DeepSeek, ChatGPT, Grok и Mistral. Особенности DeepSeek включают потоковую генерацию ответов, демонстрацию процесса «размышлений» и возможность отмены запроса.

Интерактивный интерфейс

Легко копируйте сообщения, перезапрашивайте ответы и переключайтесь между вариантами ответа для оптимального общения с AI.

Управление шаблонами

Создавайте, редактируйте и применяйте системные промпты для задания контекста AI.

Темная и светлая темы

Мгновенное переключение между темами для комфортного использования в любое время суток.

Технологии

Dart & Flutter

Основной язык программирования и фреймворк для создания кроссплатформенного приложения.

HTTP и потоковая передача

Использование пакета http для работы с потоковыми данными (SSE) и API AI-моделей.

Локальное хранилище

Сохранение чатов, сообщений, шаблонов и API-ключей на устройстве с помощью shared_preferences.

Асинхронность

Активное использование Future, Stream, async/await для неблокирующего UI и работы с сетью.

Как пользоваться

  1. Нажмите на иконку ➕ (или note_add) для создания нового чата.
  2. При необходимости выберите шаблон из списка, чтобы задать контекст диалога (системный промпт).
  3. Выберите нужную AI-модель (например, DeepSeek) из выпадающего списка.
  4. Введите ваше сообщение в поле ввода и нажмите иконку отправки (➤ или send).
  5. Взаимодействуйте с ответом: копируйте (📄), перезапрашивайте (🔄) или переключайтесь между вариантами (< X / Y >).
  6. Для DeepSeek: раскройте/сверните блок "раздумий" над ответом, чтобы увидеть процесс генерации.
  7. Управляйте чатами через боковое меню: переключайтесь между диалогами или удаляйте ненужные (долгое нажатие для выбора).
